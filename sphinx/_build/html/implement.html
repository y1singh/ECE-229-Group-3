drop_null_vals(df, axis='both', subset=[])¶

Drops columns with all nan values from a given data frame.

Parameters

  • df (pandas.DataFrame) – DataFrame for which columns are to be dropped.

  • axis (str) – Drops all rows with nan if axis=rows, all columns if axis=columns, and both if axis=both.

  • subset (list) – For all columns in subset, remove the NaN rows.

Returns

DataFrame without NaN

cal_pval.py¶

This script downloads the data for the stocks in ticker_industry_map.json. It then calculates the p-value of each stock w.r.t. the spread of COVID and stores it as a csv in p_val_march.csv

China_industry_data.py¶

This script downloads all the Chinese stock market data and stores the mapping of industry type and stock in data/china_industry_map.p and also stores the individual stock’s timeseries data in data/china_stocks_timeseries.csv.
